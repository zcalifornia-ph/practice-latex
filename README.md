## About The Project

`practice-latex` is a learning repository focused on practical LaTeX usage:

- LaTeX markup fundamentals
- package usage patterns
- document formatting and typesetting workflow

This repository is intended for iterative practice, documentation, and reproducible examples while building confidence with technical writing in LaTeX.

## Getting Started

To get a local copy and start practicing:

### Prerequisites

* Git
* A TeX distribution (`TeX Live` or `MiKTeX`)
* Any text editor (VS Code, TeXstudio, Neovim, etc.)

Verify your TeX installation:

```sh
pdflatex --version
```

### Installation

1. Clone the repository.
   ```sh
   git clone https://github.com/zcalifornia-ph/practice-latex.git
   cd practice-latex
   ```
2. Create or edit `.tex` files in your preferred learning folder.
3. Compile with `pdflatex`:
   ```sh
   pdflatex your-file.tex
   ```

## Usage

Example minimal LaTeX file:

```tex
\documentclass{article}
\begin{document}
Hello, LaTeX!
\end{document}
```

Compile:

```sh
pdflatex hello.tex
```

Use this repository to iterate through:

- text formatting
- math mode and equations
- figures and tables
- references and bibliography
- package-specific experiments
